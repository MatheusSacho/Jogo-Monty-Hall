# Jogo das Portas

Bem-vindo ao **Jogo das Portas**! Este é um jogo interativo baseado no famoso "Problema de Monty Hall", onde você pode testar suas habilidades em um jogo de escolha e estratégia.

## Descrição

O Jogo das Portas é um jogo simples onde o jogador deve escolher entre três portas. Atrás de uma das portas há um prêmio e atrás das outras duas portas há itens sem valor. O jogador faz uma escolha inicial e, dependendo do modo selecionado, pode optar por permanecer com sua escolha ou trocar para outra porta.

### Modos de Jogo

1. **Modo Livre**: O jogador faz uma escolha inicial e tem a opção de trocar de porta ou permanecer com sua escolha. É possível avançar para a próxima rodada ou reiniciar o jogo a qualquer momento.
2. **Modo Automático 1**: O jogo faz uma escolha inicial automaticamente e o jogador deve decidir se deseja permanecer com essa escolha.
3. **Modo Automático 2**: O jogo faz uma escolha inicial automaticamente e o jogador deve trocar para a outra porta automaticamente.

## Funcionalidades

- **Escolha entre 3 portas**.
- **Modos de jogo livre e automáticos**.
- **Relatórios de resultados em tempo real**.
- **Botões para trocar, permanecer, avançar para a próxima rodada e voltar ao menu**.

## Como Jogar

1. Escolha o modo de jogo desejado na tela inicial.
2. No modo livre, escolha uma porta e decida se deseja trocar ou permanecer.
3. Nos modos automáticos, o jogo fará escolhas automaticamente, e você verá o resultado ao final de cada rodada.
4. Use os botões disponíveis para interagir com o jogo conforme o modo selecionado.
